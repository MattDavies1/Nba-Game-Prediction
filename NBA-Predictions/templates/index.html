Model Explanation

ELO: Represents a team's performance relative to it's peers. This value, relative to another team, represents the probability that one team will beat another, and the most likely points spread. It considers only the outcome of every game and which team was home to calculate.

Tree: Built using sklearn Random Forest Classification. A random forest consists of a large number of individual decision trees that operate as an ensemble. Each tree in the forest predicts a class prediction (home team win or loss) based on major NBA statistics. And the class with the most "votes" becomes the overall model prediction. The inputs to the model are the rolling averages for each team leading up to their game. The model is trained and tested on 2020-21 game data.
